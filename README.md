# Netease Watcher
实时监听网易云音乐当前播放歌曲和播放进度，理论兼容`3.x`所有版本

## 使用方法
1. 启动程序
2. 连接至程序显示的WebSocket地址，即可获取当前播放歌曲和进度

### 修改监听地址
指定环境变量：
- `HOST`: 监听地址（默认`0.0.0.0`）
- `PORT`: 监听端口（默认`3574`）

### 常见问题

#### 网易云音乐主窗口会未响应
由于网易云音乐最小化一段时间后会导致数据库停止更新，故该程序会修改网易云音乐最小化行为，会偶发此BUG，目前还未修复

## 编译

1. 安装Rust
2. 运行`cargo build`

注：如不需要文字UI，则运行`cargo build --no-default-features`
